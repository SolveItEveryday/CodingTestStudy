## 📐 2. 커밋 규칙

### 브랜치 양식

- `본인 영문 이름 - 문제 이름`로 브랜치 생성 후 문제 풀이

### 커밋 메시지 양식

- `본인이름: 문제이름`
  - ex) `박장훈: 올림픽`

### PR 양식

- `PR 템플릿.md`에 맞추어 내용 작성
